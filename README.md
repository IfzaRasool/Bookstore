# Book store

> "Book store" is a Single Page App (SPA) that allows users to add books with the author name and update the book progress.

## Built With

- React

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- node and npm

### Setup

- git clone repo

### Install

- npm install

### Usage

- localhost:3000

### Run tests

npm run test
